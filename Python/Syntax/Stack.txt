class Stack:
  def __init__(self):
    self.stack = []

  def push(self, element):
    self.stack.append(element)

  def pop(self):
    if self.isEmpty():
      return "Stack is empty"
    return self.stack.pop()

  def peek(self):
    if self.isEmpty():
      return "Stack is empty"
    return self.stack[-1]

  def isEmpty(self):
    return len(self.stack) == 0

  def size(self):
    return len(self.stack)

# Create a stack
myStack = Stack()

myStack.push('A')
myStack.push('B')
myStack.push('C')

print("Stack: ", myStack.stack)
print("Pop: ", myStack.pop())
print("Stack after Pop: ", myStack.stack)
print("Peek: ", myStack.peek())
print("isEmpty: ", myStack.isEmpty())
print("Size: ", myStack.size())

'''
Class name: Stack  --> A blueprint that defines how your stack behaves and what data it holds.
Object name: myStack --> This line creates (instantiates) one actual stack from the blueprint (Stack).
                         myStack is now a real object — it has its own data and can use all the class methods.
Attribute name: stack --> It’s a variable belonging to each object of the class.

| Concept              | Example in Code     | What it Represents                     |
| -------------------- | ------------------- | -------------------------------------- |
| **Class**            | `class Stack:`      | Blueprint for creating stack objects   |
| **Object**           | `myStack = Stack()` | One real instance (your actual stack)  |
| **Attribute**        | `self.stack = []`   | Data (list) that belongs to the object |
| **Access Attribute** | `myStack.stack`     | Access the list inside that object     |

'''
